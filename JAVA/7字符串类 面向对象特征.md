# String类
Java.lang提供了3个字符串了
- String: 不可变类
- StringBuilder：可变类
- StringBuffer：可变类，除此之外，是synchronized，是线程安全的（当有多个线程并发操作这个数组的时候，不会出现错乱（会上锁）（但可能降低并发度））
（StringBuilder和StringBuffer可以toString,当然大多数时候用的还是String)

查找和匹配：
`public boolean matches(String regex)`其中regex表示regular expression(正则表达式)

# 面向对象特征
## 继承
```Java
public static void main(String args[]){
	A a = new A();
	B b = new B();
	
	a=b; // Always True
	b=a; // Conditionally True
	b=(B)b; // 强制类型转换	
	
	C c = new C();
	b=(B)c // 报错：强制也不行
}

class A{

}

class B extend A{

}

class C extend A{

}
```

**类与类的关系：**
- **is a**（子类）
- **has a**
	- **association**:联系，一个类的引用是另一个类的成员变量
	- **aggregation**:更强的联系，在那个类中不需要考虑另一个类
	- **composition**:最强的联系（组合），类离开被引用的类就没有意义
- **Dependency（依赖关系）**:方法的参数，返回值的要被另一个类使用，如果删除这个类，这另一个类通不过编译
## 多态
**子类的构造函数默认包含`super()`（父类的无参构造函数）**
整个构造函数的调用链（从这个类到object）会自动延伸
而对于抽象类也可以有构造函数，但是这个构造函数只能被子类的`super()`调用
**面向对象原则：** ==组合优于继承==

## Final关键字
```Java
public final class A{// 不能再被继承
	public final void function(){
	}// 这个函数在子类继承父类时不会被继承
}
```
